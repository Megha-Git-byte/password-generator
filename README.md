# Password Generator

A customizable Python command-line application that generates strong and secure passwords. Users can specify the desired password length and include different character types such as uppercase letters, lowercase letters, numbers, and special symbols. The tool helps create random passwords that improve account security and reduce the risk of unauthorized access.

## Features

* Generate secure random passwords
* Customize password length
* Include letters, numbers, and special characters
* Simple command-line interface
* Lightweight and easy to use

## Technologies Used

* Python 3
* Built-in Python modules for random password generation

## How to Run

1. Clone the repository.
2. Open the project folder in VS Code.
3. Run:
   python pypassword_generator.py
